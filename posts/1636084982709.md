List of things to consider when writing node-express api:

- bodyparser
- static files
- auth
- error handler
- authorisation middleware
- cors
- sanitize
- xss
- rate limiting
- dev logger morgan
- validator
- hash
- file uploads
- testing (jest)
- async handler

Install express deps:
```bash
npm i express dotenv morgan cors
```
